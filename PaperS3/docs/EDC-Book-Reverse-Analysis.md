# EDC Book 固件逆向分析报告

**固件**: EDC Book (梦西游/mengxiyou) v3.x  
**来源**: M5Burner CDN  
**大小**: 16MB  
**芯片**: ESP32-S3 (从 strings 确认)  
**分析时间**: 2026-04-25

---

## 1. 固件结构

### 分区表

| 分区 | 类型 | 偏移 | 大小 | 说明 |
|------|------|------|------|------|
| nvs | DATA | 0x9000 | 20K | NVS 存储 |
| otadata | DATA | 0xe000 | 8K | OTA 升级数据 |
| **app0** | **APP** | **0x10000** | **5MB** | **主程序** |
| **font** | **DATA** | **0x510000** | **10MB** | **字体分区** |
| coredump | DATA | 0xf10000 | 128K | 崩溃转储 |
| crashlog | DATA | 0xf30000 | 32K | 崩溃日志 |
| spiffs | DATA | 0xf38000 | 800K | LittleFS 文件系统 |

### 关键发现
- **字体分区高达 10MB**：说明内置了完整字库，或者预留了大字库存储空间
- **LittleFS 800KB**：存放 Web 配置界面文件（已提取到 CSS/HTML）
- **APP 分区 5MB**：相当大的程序，功能丰富

## 2. 功能特征（从字符串提取）

### 2.1 书籍格式支持
- **TXT**: 支持 UTF-8/GBK/UTF-16LE/UTF-16BE/BIG5 编码
- **EPUB**: 已确认支持（从 M5Burner 描述和竞品分析确认）
- **目录结构**: `/book` 目录，支持子目录

### 2.2 字体系统
- **内置字体**: 自带默认字体（在 10MB font 分区）
- **外部字体**: 支持 6 种自定义字体 + 1 种默认字体
- **字体格式**: `.bin` 格式
- **字体目录**: `/font`
- **字号**: 支持大/小字号切换，"保留原来较大的字体样式为小档"

### 2.3 翻页系统
- **触摸翻页**: 点击前 1/3 上一页，后 2/3 下一页
- **翻页特效**: "模拟书页滚动的翻页效果"
- **快速翻页**: "可以快速连续翻页"
- **滑动翻页**: "翻页滑动 dir=%d ori=%d scrollDir=%d isPre=%d"
- **章节分页**: "保证新章节始终出现在下一页"

### 2.4 蓝牙翻页器
- **HID 设备支持**: 完整的蓝牙 HID 键盘/遥控器支持
- **自动重连**: "重连] 异步连接设备"
- **电量读取**: "订阅] 蓝牙设备电量: %d%%"
- **Report Map**: 完整的 HID Report Descriptor 解析
- **超时处理**: "蓝牙翻页器超时未连接"
- **省电**: "则自动关闭蓝牙功能以节省电量"

### 2.5 WiFi 传书
- **双模式**: AP 模式（配置密码）+ STA 模式（连接 WiFi）
- **Web 配置**: 内置完整的 HTTP 服务器和 Web UI
- **目录**: 支持目录浏览和创建
- **在线字体设置**: "可在在线配置的/font目录下设置某一个外部字体为默认字体"

### 2.6 封面与锁屏
- **封面**: `/cover` 目录，支持 png/jpg
- **书籍封面**: 优先显示书籍同目录下同名图片（如 `/book/edcbook.txt` 的封面为 `/book/edcbook.png`）
- **锁屏**: "书籍锁屏"、"封面锁屏"
- **壁纸**: `/bg` 目录，"/pic" 目录随机显示
- **老板键**: "老板键壁纸和书籍封面"
- **低电量提醒**: "关机将优先显示低电量提醒图片"

### 2.7 书签系统
- **10 个书签**: 壹、贰、叁、肆、伍、陆、柒、捌、玖、拾
- **书签文件**: `Bookmarks` 相关存储

### 2.8 阅读设置
- **排版优化**: "对TXT文字的排版进行相关的优化"
- **页面布局**: 上下边距、左右边距、行间距、字间距、段间距
- **翻页方式**: 可修改翻页方式
- **全刷周期**: "越清晰刷新速度越慢"、"墨水屏全屏刷新的频次"
- **自动旋转**: 支持
- **继续阅读**: 打开自动回到上次位置

### 2.9 最近阅读
- **recents.ini**: 存储最近阅读列表
- "最近阅读文件已删除"、"最近阅读文件打开失败"

### 2.10 章节系统
- **章节缓存**: "章节缓存命中"
- **章节索引文件**: 自动生成和加载
- "从%s加载了%d个章节"

## 4. 技术栈推测

| 组件 | 推测 |
|------|------|
| 框架 | ESP-IDF（不是 Arduino） |
| 文件系统 | LittleFS（不是 SPIFFS/FAT） |
| 网络 | 自建 HTTP 服务器 |
| 蓝牙 | NimBLE（从日志字符串确认） |
| 字体 | 自定义二进制点阵 |
| 渲染 | 直接操作 EPD framebuffer |

## 6. 关键启示

### 6.1 字体策略
EDC Book 的 **10MB 字体分区** 是最大亮点。这意味着：
- 出厂自带完整中文字库，用户无需准备字体
- 支持 7 种字体切换（6 自定义 + 1 默认）
- 字库是 `.bin` 格式，可能和我们的格式兼容

**建议**：我们可以考虑内置一个小字库到固件中，避免用户首次使用无字库的困境。

### 6.2 Web 配置界面
EDC Book 有完整的 Web UI（从 CSS 提取确认），支持：
- 文件管理器
- 字体在线切换
- WiFi 配置
- 阅读设置

**建议**：我们的 WiFiUploader 可以扩展成完整的 Web 文件管理器。

### 6.3 翻页特效
"模拟书页滚动的翻页效果" — 这是墨水屏上的高级功能，需要局部刷新波形控制。

### 6.4 目录支持
支持子目录浏览，我们的 FileBrowser 目前是单层。

### 6.5 编码支持
EDC Book 支持 UTF-16LE/UTF-16BE/BIG5，我们只支持 UTF-8/GBK。
